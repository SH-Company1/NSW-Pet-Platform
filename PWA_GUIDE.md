# PWA (Progressive Web App) 가이드

## 🎉 앱처럼 사용하기

NSW Pet Platform은 PWA로 설정되어 있어서 **모바일 브라우저에서 앱처럼** 사용할 수 있습니다!

---

## 📱 iOS (iPhone/iPad)에서 홈 화면에 추가

### 방법 1: Safari 브라우저
1. Safari에서 웹사이트 접속
   ```
   https://nsw-pet-platform.vercel.app
   ```

2. 하단 **공유 버튼** (□↑) 탭

3. **"홈 화면에 추가"** 선택

4. 앱 이름 확인 → **"추가"** 탭

5. 홈 화면에 앱 아이콘 생성! 🎉

### 사용
- 홈 화면의 아이콘을 탭하면 전체 화면으로 실행
- 주소창, 탭바 없음 (네이티브 앱처럼)
- 백그라운드 전환 가능
- 멀티태스킹 지원

---

## 📱 Android에서 홈 화면에 추가

### 방법 1: Chrome 브라우저
1. Chrome에서 웹사이트 접속
   ```
   https://nsw-pet-platform.vercel.app
   ```

2. 우측 상단 **⋮** (메뉴) 탭

3. **"홈 화면에 추가"** 또는 **"앱 설치"** 선택

4. 팝업에서 **"추가"** 또는 **"설치"** 탭

5. 홈 화면에 앱 아이콘 생성! 🎉

### 방법 2: 자동 설치 배너
- 일부 Android 기기에서는 사이트 접속 시
- 하단에 "앱 설치" 배너가 자동으로 표시됨
- "설치" 버튼만 누르면 끝!

---

## 💻 데스크톱 (PC/Mac)에서 설치

### Chrome / Edge
1. 주소창 우측의 **설치 아이콘** (⊕) 클릭
2. "설치" 클릭
3. 독립 창으로 실행됨

### Safari (Mac)
- 파일 → "이 페이지를 Dock에 추가"
- Dock에서 클릭하면 독립 창으로 실행

## ✨ PWA의 장점

### 1. 앱처럼 작동
- ✅ 전체 화면 모드
- ✅ 홈 화면 아이콘
- ✅ 스플래시 스크린
- ✅ 앱 전환기에 표시

### 2. 설치 불필요
- ✅ 앱스토어 방문 불필요
- ✅ 용량 작음 (네이티브 앱 대비)
- ✅ 자동 업데이트
- ✅ 스토어 심사 불필요

### 3. 오프라인 지원 (향후 추가 가능)
- 네트워크 없이도 기본 화면 표시
- 데이터 캐싱

### 4. 푸시 알림 (향후 추가 가능)
- 웹에서도 푸시 알림 가능
- 사용자 재참여 유도

## 🎨 PWA 설정

### 현재 설정된 내용
```json
{
  "name": "NSW Pet Platform",
  "short_name": "NSW Pet",
  "theme_color": "#4CAF50",
  "background_color": "#FFFFFF",
  "display": "standalone",
  "orientation": "portrait"
}
```

### display: "standalone"
- 브라우저 UI (주소창, 탭바) 숨김
- 네이티브 앱처럼 전체 화면 사용
- 상태바만 표시

### theme_color: "#4CAF50"
- 상단 상태바 색상
- 앱 전환기에서 보이는 색상
- 브랜드 컬러 적용

## 🔍 PWA 작동 확인

### Chrome DevTools로 테스트
1. 웹사이트 접속
2. F12 (개발자 도구)
3. **Application** 탭
4. **Manifest** 섹션 확인
   - Name, Icons, Theme Color 등 확인
5. **Lighthouse** 탭
   - "PWA" 카테고리 실행
   - 점수 확인

### 확인 항목
- ✅ manifest.json 존재
- ✅ 아이콘 설정
- ✅ theme_color 설정
- ✅ HTTPS 사용 (Vercel 자동 제공)
- ✅ 반응형 디자인

## 📊 PWA vs 네이티브 앱 비교

| 기능 | PWA | 네이티브 앱 |
|------|-----|------------|
| 설치 | 브라우저에서 즉시 | 앱스토어 필요 |
| 용량 | 매우 작음 | 큼 (50MB+) |
| 업데이트 | 자동 (새로고침) | 수동 (스토어) |
| 오프라인 | 제한적 지원 | 완전 지원 |
| 성능 | 우수 | 최고 |
| 카메라/센서 | 제한적 | 완전 접근 |
| 배포 | 즉시 | 심사 필요 |
| 비용 | 무료 (호스팅만) | 개발비용 높음 |

## 🎯 사용 시나리오

### 일반 사용자
1. 모바일 브라우저로 접속
2. "홈 화면에 추가"
3. 홈 화면 아이콘으로 바로 접속
4. 앱처럼 사용 ✨

### 테스터
- 앱스토어 배포 전에 PWA로 먼저 테스트
- URL만 공유하면 즉시 테스트 가능
- 피드백 받고 빠르게 수정

### 개발자
- 빠른 배포 및 업데이트
- 크로스 플랫폼 (iOS, Android, Desktop)
- 스토어 수수료 없음

---

## 🚀 배포 후 확인사항

### Vercel 배포 완료 후
1. 모바일에서 사이트 접속
2. "홈 화면에 추가" 가능한지 확인
3. 설치 후 전체 화면으로 작동하는지 확인
4. 상태바 색상 (#4CAF50) 확인

### 문제 해결
- 아이콘이 안보이면: `public/` 폴더에 아이콘 파일 확인
- 설치 안되면: HTTPS 사용 여부 확인
- 전체 화면 안되면: manifest.json의 display 설정 확인

---

## 💡 추가 개선 아이디어

### 향후 추가 가능 기능
1. **Service Worker**
   - 오프라인 지원
   - 백그라운드 동기화
   - 푸시 알림

2. **App Shortcuts**
   - 앱 아이콘 길게 눌러 빠른 실행
   - 주요 기능 바로가기

3. **Share Target**
   - 다른 앱에서 공유 시 NSW Pet으로 선택 가능

4. **Badging API**
   - 앱 아이콘에 알림 배지 표시
